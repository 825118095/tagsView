# tagsView 

后台页面单独出来的tagsView功能，用于学习使用。

## 如何安装与使用

npm install //安装依赖

npm run dev //服务端运行

npm run build  //项目打包

## 技术栈

```
- Vue（核心框架）
- Vue-Router（页面路由）
- Vuex（状态管理）
- Less（CSS预处理器）
- element-ui(ui框架)
```
